# Employee Management Dashboard

## Project Overview
This is a simple React-based Employee Management Dashboard built as part of a frontend assessment.

The application allows users to:
- Login using mock authentication
- View a dashboard with employee summary
- Add, edit, delete employees
- Upload profile image with preview
- Search and filter employees
- Toggle active/inactive status
- Print employee list

All data is stored in browser LocalStorage (no backend).
